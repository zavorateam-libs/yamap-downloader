# YaMap 3D Viewer

Приложение для просмотра и загрузки 3D-моделей зданий с Яндекс.Карт в формате GLB.

## КАК использовать

### Установка

**Для Windows:**
1. Скачайте последний релиз в формате `.exe`
2. Запустите портативный исполняемый файл

**Для Linux:**
1. Скачайте файл `.AppImage`
2. Сделайте его исполняемым: `chmod +x YaMap-3D-Viewer-x.x.x.AppImage`
3. Запустите двойным кликом или из терминала: `./YaMap-3D-Viewer-x.x.x.AppImage`

### Работа с приложением

1. **Загрузка реестра моделей:**
   - Вставьте URL к JSON-файлу с реестром моделей в поле "Registry (JSON)"
   - Приложение автоматически загрузит список доступных моделей

2. **Просмотр моделей:**
   - Выберите интересующую модель из списка
   - Модель отобразится в основном окне просмотра
   - Используйте мышь для вращения, масштабирования и перемещения

3. **Настройка отображения:**
   - Измените тему отображения в поле "Render Theme Suffix" (например, "dark", "light")
   - Используйте меню "Визуальные эффекты" для применения стилей (PS1 Retro, CRT Display)

4. **Сохранение моделей:**
   - Выберите модель для просмотра
   - Используйте пункт меню "Файл" → "Скачать текущую модель"
   - Выберите место сохранения файла в формате GLB

## ЗАЧЕМ создан

Этот проект создан для:

- **Исследования 3D-данных Яндекс.Карт** - предоставляет удобный способ просмотра и анализа 3D-моделей зданий
- **Сохранения моделей** - позволяет загружать модели в формате GLB для дальнейшего использования
- **Изучения архитектуры зданий** - дает возможность детально рассмотреть 3D-модели зданий с разных ракурсов
- **Разработки контента** - предоставляет доступ к качественным 3D-моделям для использования в своих проектах
- **Образовательных целей** - помогает понять, как устроены 3D-модели в современных картографических сервисах

Проект использует официальное API Яндекс.Карт для получения 3D-моделей, обеспечивая актуальность и высокое качество данных.

## ФИШКИ проекта

### Основные функции:

- **Просмотр 3D-моделей** - отображение моделей зданий с Яндекс.Карт в высоком качестве
- **Загрузка моделей** - сохранение моделей в формате GLB для дальнейшего использования
- **Поддержка тем** - возможность просмотра моделей в разных темах оформления
- **Реестр моделей** - работа с внешними JSON-файлами, содержащими списки моделей
- **Локальные файлы** - возможность загрузки локальных реестров моделей

### Визуальные эффекты:

- **PS1 Retro** - стилизация под визуальные эффекты PlayStation 1
- **CRT Display** - имитация ЭЛТ-монитора с эффектом мерцания и сканирующими линиями
- **Настройка качества** - регулировка параметров рендеринга для оптимизации производительности
